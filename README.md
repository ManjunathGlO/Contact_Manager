<h1 align="center">📇 Contact Management System</h1>

<p align="center">
A simple and efficient Contact Management Application built using <b>Python Flask</b>, <b>SQLite</b>, and <b>Bootstrap</b>.
</p>

---

<h2>🚀 Features</h2>

<ul>
  <li>➕ Add new contacts</li>
  <li>📋 View all contacts</li>
  <li>✏️ Update existing contacts</li>
  <li>❌ Delete contacts</li>
  <li>🔍 Search contacts by name</li>
  <li>✅ Email validation</li>
  <li>🚫 Duplicate email protection</li>
  <li>💾 Persistent storage using SQLite</li>
</ul>

---

<h2>🛠️ Tech Stack</h2>

<ul>
  <li><b>Backend:</b> Python (Flask)</li>
  <li><b>Database:</b> SQLite</li>
  <li><b>Frontend:</b> HTML, CSS, Bootstrap</li>
  <li><b>Deployment:</b> Render</li>
</ul>
